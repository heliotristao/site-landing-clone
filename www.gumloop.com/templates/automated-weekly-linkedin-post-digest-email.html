DESCRIPTION

Overview

LinkedIn has a lot of noise and I got tired of wading through their messy feed to see the content from people I actually want to see... so I did something about it.

This workflow takes a list of LinkedIn Profile URLs from a Google Sheet that you actually want to stay up to date with and it sends an email with all their posts.

It's setup to scrape the posts from last week so I recommend setting the trigger for once a week in Gumloop.

When to Use

Ideal for weekly or periodic triggers that compile recent LinkedIn content and email it to a specified list.

Integrations

  • Google Sheets: Retrieves LinkedIn Profile URLs from a specified spreadsheet.

  • Apify: You'll need an Apify account (a free one works) to do the LinkedIn scraping.

Output

Generates an HTML-formatted email digest summarizing LinkedIn posts and sends it to the email(s) you set in the Gmail Sender node.

HOW DO YOU SET THIS UP?

1.

Configure Google Sheet URL and Range

Enter the URL of your Google Sheet containing the LinkedIn Profile URLs in the Google Sheets node.

2.

Get Your Apify API Token

In the LinkedIn URL to HTML subflow you'll need to add your Apify API to the Call API node URL that's in there. Just and FYI, don't share this workflow to anyone you don't want to have access to your Apify API key.

3.

Set Email Recipients

Ensure the email field in the sending node is mapping correctly to the addresses you want to send to. Make sure HTML sending is checked. If you want to change the look of the Email I'd recommend dumping this code into Gummie or ChatGPT and asking it to output new sample HTML code.
