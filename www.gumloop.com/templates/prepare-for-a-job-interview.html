DESCRIPTION

This agent acts like a job interview prep coach. Give it the URL of a job posting, and it will analyze the job posting, compare it against the candidate's LinkedIn profile, and return a report with specific recommendations on how to prepare for an interview for that job posting. If you'd like, the agent can also email the report to you for later review.

To customize this agent for your own use, replace the LinkedIn profile URL in the "Template Instructions" portion of the instructions with the URL of your own LinkedIn profile.

This agent uses a web scraping Gumloop flow; creating a copy of this agent will also create a copy of the web scraping Gumloop flow.

HOW DO YOU SET THIS UP?

1.

Authenticate with Gmail

2.

Modify the Template Instructions to contain your LinkedIn profile URL

Replace the bracketed text ("[https://www.linkedin.com/in/example/]") with the URL of your LinkedIn profile.
